### About

Wholphin offers an alternative Jellyfin client experience for Android TV. The interface and navigation patterns are designed from the ground up, influenced by Plex while remaining focused on clarity and speed. Playback uses Media3 / ExoPlayer to ensure consistent, reliable performance.

> “Never half-phin two jellies. Always wholphin one jelly.”

---

### Key Features

- Clean navigation drawer available throughout the app
- Clear library browsing with visible titles
- Optional theme music playback for TV series
- Playback controls aligned with standard Android TV expectations
  - D-Pad seeking
  - Quick chapter selection
  - Optional rewind-on-resume
- Visual consistency improvements
  - Progress hints during seek
  - Poster-focused Continue Watching and Next Up

### Installation

**Downloader Code:** `8668671`

1. Enable installation from unknown sources.
2. Install using one of these:
   - Use the Downloader app and enter `8668671`
   - Download the latest APK from Releases and transfer using:
     - USB storage
     - Send Files to TV
     - ADB (`adb install wholphin.apk`)

Once installed, updates can be triggered from within the app. The first update may request permission to allow Wholphin to update its own APK.
